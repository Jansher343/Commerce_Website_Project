Overview

UMT Store is a simple e-commerce website featuring a product catalog, contact form, and order tracking. The site is designed with a clean layout and user-friendly interface. It includes various sections such as an announcement bar, navigation bar, product display, and a contact form.

Features

Announcement Bar: Displays key promotions and announcements.
Navigation Bar: Links to Home, Catalog, Contact, and Track Your Order pages.
Product Display: Showcases products with details and pricing.
Contact Form: Allows users to send inquiries directly via email.
Search Functionality: Search for products (note: search functionality is not implemented).
